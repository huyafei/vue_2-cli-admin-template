# vue_2-cli-admin-template
该模版是从 [vensst-cli](https://huyafei.github.io/yfhu-blog/cli/) 工具中下载

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles and minifies for production
```
// 生产测试环境
npm run build:uat

// 生产正式环境
npm run build
```

### Lints and fixes files
```
npm lint
```
